3.1 添加 Action 会有一系列的要求  
1. Action 的 path ，必须要以 gradle中配置的 模块名称+"/" 开头，按照提示修改就行
2. Action 必须要实现 IRouterAction，按照提示修改就行
3. 所有执行代码都写在 invokeAction 里面

3.2 在需要跨模块通信的lib里面配置   
3.3 在任意模块都可以执行跳转
